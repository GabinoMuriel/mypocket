🚀 Project Status: MyPocket (Personal Finance Tracker)

Styling: Tailwind CSS v4 + shadcn/ui.

Backend: Supabase (Auth, DB, Storage).

State Management: Zustand with persist middleware.

Folder Structure: Service-oriented architecture (src/services, src/lib, src/store).

Linguistic Rule: English for code/logic, Spanish for UI/messages/validation.
